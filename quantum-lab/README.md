# Schrödinger Cat Paradox

A simple cartoon browser game that reconstructs Schrödinger's cat thought experiment.

## Run locally

Open `index.html`, or serve the directory:

```bash
python3 -m http.server 4173 --directory quantum-lab
```

Then visit `http://127.0.0.1:4173`.

## Learning goal

Players assemble the radioactive atom, detector, hammer, flask, and cat; seal the chamber; wait; and measure one outcome. The game explains probability, measurement, and the purpose of Schrödinger's criticism without claiming that real cats literally occupy both everyday states.
